Unit 2 Study Guide

Chapter 4

  • Website: A collection of webpages that can be accessed online.
  • Canva: Can be used as a website design software.
  • Mockup: A visual representation of the design of a website, software, or other product.
  • HTML: Stands for Hyper Text Markup Language; a programming lanuage that was created in 1993 by Tim-Berners Lee.
  • JavaScript: A web programming language used for providing behavior to a webpage.
  • Design: The process of making a product or service that meets the need or want of the customer.
  • Design Process: A formal approach used by architects, engineers, and scientists for breaking down a large project into manageable chunks.
  • Design Specification: A detailed and precise proposal or plan for a particular design.
  • Gig Platform: Web or app-based marketplaces that connect businesses with providers of short-term, on-demand services.
  • src: To specify the URL of a file.
  • Gig Economy: A term used to describe the growing trend of workers participating in short-term, project-based work.
  • Script: A block of code that is used to execute a script, like JavaScript.
  • Web Browser: A software application for accessing information on the World Wide Web.
  • HTML Comments: Pieces of syntax used to document code.
  • File Extension: A way of identifying a particular type of file.

Chapter 5

  • Rules for naming variables: Must start with a letter, an underscore or a dollar sign. Cannot contain a space. Cannot be the same as reserved keywords such as if or const. Written in camelCase. Should be given descriptive names that indicate their content and usage. Do not have set types, it can be useful to include an indication of the type in the name.
  • Variable: A named container for a value that can change. In programming, variables are assigned values and used to store data. The value can be a number, text, or a combination of both.
  • Reserved Word: A word that can not be used to name a variable, function, method, or property. They are “reserved” by the program for specific purposes.
  • Data: A collection of symbols that convey information.
  • Data Type: A classification that tells a computer how to store and manipulate a particular kind of information. There are many different data types, but the most common are text, numbers, and booleans.
  • Number: A mathematical concept that refers to the quantity of something.
  • String: A data type, they are text and are written with double quotes surrounding them. They are stored inside of variables.
  • Assignment Operator: It’s the equal sign. It is used to assign a value to a variable.
  • Syntax: The set of rules that govern the structure of sentences in a language.
  • Semantics: The study of the meaning of words and phrases in a language. It is a process of assigning meaning to the symbols of a language in order to create and interpret sentences.
  • Concatenation: The process of joining two strings together. In JavaScript, concatenation is performed by “adding” two strings together using the + operator.
